# SERVER

## _Make commands_

#### **Simple compilation**
Type the following :
```
    make
    ./server <port>
```

#### **Compile + run**
Type the following :
```
    make run
```

#### **Compile + run with valgrind**
Type the following :
```
    make valrun
```

#### **Initialize data base**
Type the following :
```
    make init_db
    ./init_db
```

#### **Clean**
```
    make clean
```
